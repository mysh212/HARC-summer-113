## **A. 黑閃爆炸** ***<font color = '#AAAAAA'> BF explode </font>***

`time limit` 1s
`memory limit` 256MB

### ***Statement***
揍術迴戰的主角唬障為了打敗宿攤

需要透過擊出多次黑閃恢復輸出效率

每擊出一次輸出效率就會翻倍

唬障原本的輸出效率會是1點

這時如果打出3發黑閃輸出效率會是8點

唬障共會打出 $n$ 發黑閃

由於唬障與宿攤已經從上次 $HARC$ 線上賽打到現在

n變得極大 $0 \leq n \leq 10^{10^{8}}$

但宿攤在最後打出世界斬 

將其輸出效率模除10

如原本16 -> 6

請你算出唬障最後的輸出效率 $Ans$




### ***Input***
$n$
### ***Output***
$Ans$


<div class='page'>



### ***Sample Input 1***
```
5
```


### ***Sample Output 1***
```
2
```

### ***Sample Input 2***
```
100
```


### ***Sample Output 2***
```
6
```

### ***Note***
$0 \leq n \leq 10^{10^{8}}$
你該不會以為是假的吧.w.
